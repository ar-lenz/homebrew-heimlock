# Heimlock DMGs

macOS DMGs are distributed as **GitHub Release assets** (the Electron build is ~205 MB, over GitHub's 100 MB in-repo file limit).

Download the latest directly: https://github.com/ar-lenz/homebrew-heimlock/releases/latest

Or via Homebrew: `brew install --cask ar-lenz/heimlock/heimlock`

The older `Heimlock-0.6.0`/`0.7.0` DMGs here are legacy Tauri builds.
